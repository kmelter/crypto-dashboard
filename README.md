# Cryptocurrency Dashboard

## Purpose
A website that allows the user to search for a cryptocurrency and see its current value as well as dollar conversion rates to various other currencies.

## Built With
* HTML
* CSS
* JavaScript

## Libraries
* Material UI

## APIs
* CoinCap API
* Exchange Rates API
